#Normal App 3

Sample application to simulate normal conditions of application.

##Compile

        javac NormalApp3.java

##Run

        java NormalApp3
or

        java NormalApp3 <long> upperBound



| Options  |  Description  | Default |
| --------|---------|-------|
|***\<int> upperBound*** | Upper Bound of prime| Integer.MAX_VALUE |
